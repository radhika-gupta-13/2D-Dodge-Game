# 🎮 2D Dodge Game

A simple Unity 2D game where the player dodges falling enemies.

## 🚀 Features
- Player movement (A/D or Arrow Keys)
- Enemy spawning system
- Score tracking
- Increasing difficulty over time
- Explosion effects 💥

## 🎮 Controls
- Move: A / D or Arrow Keys
- Restart: R

## 🛠️ Built With
- Unity
- C#
